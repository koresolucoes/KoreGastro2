Como funciona?

No ChefOS, cada conta de usuário representa uma Loja Principal. Para gerenciar múltiplas lojas, você deve ser convidado pelo proprietário da outra loja.

Peça para o proprietário da outra loja acessar:

  1. Menu Configurações
  2. Aba Gestão de Equipe
  3. Adicionar seu e-mail como Gestor
{{ currentUserEmail }}

Assim que o convite for enviado, a nova loja aparecerá automaticamente na sua lista de lojas.
